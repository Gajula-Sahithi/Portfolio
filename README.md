# Deploying a Web Application on AWS EC2

## Project Overview
This project demonstrates deploying a web application on an AWS EC2 instance.

## Technologies Used
- AWS EC2
- Linux
- Git
- Nginx/Apache
- HTML/CSS/JavaScript (or React)

## Deployment Steps
1. Launch EC2 instance
2. Configure Security Groups
3. Connect via SSH
4. Install web server
5. Deploy application
6. Access through public IP
